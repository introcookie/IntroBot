## ✨ Возможности

| Категория | Описание |
|-----------|----------|
| ⚡ **Мгновенное подключение** | Боты заходят на сервер за считанные секунды. |
| 🧠 **Авто‑регистрация и вход** | Сами вводят `/register` и `/login` с единым паролем. |
| 🧵 **Последовательный запуск** | Стартуют по очереди (`User`, `User1`, `User2`…), не перегружая сервер. |
| 💬 **Массовый спам** | Отправляйте одно сообщение сразу от всех ботов. |
| 🕹️ **Anti‑AFK** | Автоматические прыжки, повороты и движение. |
| 🎛️ **Панель управления** | Список активных ботов, ручной кик, включение/выключение Anti‑AFK. |
| 📊 **Статус в реальном времени** | Количество онлайн‑ботов отображается внизу окна. |
| 🎨 **Современный дизайн** | Тёмная тема, анимированный фон, цветной лог, плавные переходы. |

## 🖥️ Интерфейс

### Главное окно

- **Левая панель**  
  `SEND BOTS` – открыть настройки запуска  
  `STOP ALL` – остановить всех ботов  
  `CLEAR LOG` – очистить лог

- **Правая панель**  
  Вкладка **Log** – цветной вывод всех действий ботов  
  Вкладка **Control** – управление активными ботами

### Окно настройки ботов

| Поле | Назначение |
|------|------------|
| **Server IP (with port)** | Адрес сервера (например `play.hypixel.net:25565`) |
| **Version** | Версия Minecraft (например `1.20.4`) |
| **Base nickname** | Префикс для имён (`User` → `User`, `User1`, `User2`…) |
| **Bot count** | Количество ботов (до 1000) |
| **Message** | Сообщение, которое боты будут отправлять после входа и периодически |
| **Delay** | Пауза между спам‑сообщениями в секундах |
